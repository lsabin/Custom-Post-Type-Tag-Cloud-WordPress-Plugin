=== Custom Post Type Tag Cloud ===
Contributors: jenleighkelly
Tags: posts, widget, taxonomy
Requires at least: 3.0
Tested up to: 3.3.1
Stable tag: 1.0

Modifies the native WordPress Tag Cloud widget to be able to choose taxonomies from all post types to render in a tag cloud format.

== Installation ==

1. Upload `custom-post-type-tag-cloud.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Through the Appearance > Widgets menu in WordPress, place the 'Custom Post Type Tag Cloud' widget in a registered sidebar.
4. Configure options and save.
